### What is FACstamp?
FACstamp is a project to help support trustable, non duplicitous human communication:
- Technically, FACstamp is a set of software applications leveraging cryptographic and public ledger technologies providing highly secure digital identity and content management
- Legally, FACstamp is a set of  software agreements dealing with the creation, publication, and follow-up communication and verification regarding digital multi media and content
- Usage wise, FACstamp supports an [Expressed Warranty](https://peoples-law.org/warranties#expresswarranties) of Fitness-for-Use that end users or applications can use to inspect Fitness-for-Use compliance of digital content that has been cryptographically stamped and signed with a FACstamp
- There are potentially significant [[Market Analysis|new markets]] that FACstamp adoption will create as it solves an impressive list of [[Problems and Solutions|fundamental usage issues]] currently plaguing the Internet

See the  [What is FACstamp?](https://docs.google.com/presentation/d/1ntCFRPgfZhdFjXoyJKOvZfSz-0Ic0FO2rrKoWm7-wuQ/edit?usp=sharing) presentation for a summary of 'What is FACstamp?'.
### Why FACstamp?
With the accelerating use of the Internet, the ability to discern accurate information delivered with virtuous intention has become difficult, time consuming, and an increasing insurmountable challenge.  A true democracy, one that is not a democracy in name only, requires the electorate to be accurately informed (accurate information) without associated duplicitous logic, rationalizations, and/or rhetoric (non virtuous manipulation and bamboozlement).  The electorate should be discerning with both individual and cultural biases.  FACstamp is a project aimed at addressing this basic difficulty. 

See the [Why FACstamp?](https://docs.google.com/presentation/d/1sz3LdS9dzUrgo_tjs2UQpdySK1FTu_T7J79z5pK4CQk/edit?usp=sharing) presentation for a summary of 'Why FACstamp?'
### What is the FACstamp Expressed Warranty of Fitness of Use?
A key aspect of FACstamp concerns the [[Express Warranty of Fitness-For-Use]] that is managed by the various FACstamp license and usage agreements.  When digital content is stamped with a FACstamp, consumers of that content have access to the [[Express Warranty of Fitness-For-Use]].  This FACstamp merchantability is defined by two primary requirements, the first [[Requirement 1]] is a historical accuracy clause and the second [[Requirement 2]] is a virtuous and duplicity free delivery clause.  See [[FACstamp Deeper Dive]] for more details.
